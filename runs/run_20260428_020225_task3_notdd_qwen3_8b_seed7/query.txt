Title: Unique Paths with Obstacles

Description:
A robot on an m x n grid starts at the top-left corner and wants to reach the bottom-right. It can only move right or down. Some cells have obstacles (marked 1), free cells are 0. Return the number of unique paths from start to end. Return 0 if start or end is blocked.

Function Signature:
def unique_paths_with_obstacles(obstacle_grid: list[list[int]]) -> int:

Constraints:
m == len(obstacle_grid), n == len(obstacle_grid[0]), 1 <= m, n <= 100

Example Inputs:
[[0,0,0],[0,1,0],[0,0,0]]

Expected Outputs:
2
