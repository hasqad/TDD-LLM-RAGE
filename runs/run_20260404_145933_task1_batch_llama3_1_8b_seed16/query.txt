Title: Molecule to Binary Code Converter

Description:
Convert a chemical formula into a binary code based on the atomic weights of its elements. The binary code is generated by summing the ASCII values of each element's symbol and converting that sum to binary.

Function Signature:
def molecule_to_binary(molecule: str) -> str:

Constraints:
molecule will be a valid chemical formula consisting only of uppercase letters (e.g., H2O, C6H12O6). ASCII values for A-Z are 65-90. The output should be the binary representation as a string with '0b' prefix.

Example Inputs:
"H2O"
"C6H12O6"

Expected Outputs:
'0b1000110'
'0b1110011000011110'
